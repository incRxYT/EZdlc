# Contributing to EZdlc

Thanks for your interest in contributing to EZdlc!

This project is a lightweight C++ CLI tool designed to help manage and organize Call of Duty: Black Ops 3 files for use with the BOIII client.

---

## Project Goals

EZdlc is focused on:

- Safe file organization and management
- CLI-based usability (no GUI)
- Clean, readable C++ code
- Cross-version compatibility for BO3/BOIII setups

We are NOT:
- distributing any game files
- modifying or cracking game content
- bypassing ownership or licensing systems
