EZdlc

A lightweight C++ CLI tool for managing Black Ops 3 files for BOIII compatibility.

View on GitHub

⚙️ What is EZdlc?

EZdlc is a command-line tool that helps organize and map Black Ops 3 game files (zone, sound, video, and all folders) into BOIII-compatible structure.

🚀 Features

  • CLI-based interface (no GUI)
  • Dry-run preview before applying changes
  • Copy or Symlink file modes
  • Safe file scanning using known DLC patterns
  • Windows-compatible C++17 implementation

🧪 Example Usage

Run EZdlc in your terminal:

EZdlc.exe

Follow prompts to select BO3 and BOIII folders, then apply changes.

🛠 Build Instructions

Using CMake:

cmake -S . -B build
cmake --build build --config Release

⚠️ Disclaimer

EZdlc does not include or distribute any game files. Users must own their game content. This tool only manages local file organization.
